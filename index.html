mdSlide is the simplest presentation creating tool, with using Markdown.

You can "write" the presentation with like a feeling of Markdown editor.

Features

  • Create presentation slides by Markdown
  • Cross-platform (Supported Windows, Mac OS X and Linux)
  • Live preview with 3 modes (Markdown, 1:1 slide and list view)
  • Export slides as PDF
  • The most simple slide theme, based on GitHub style

How to write slides?

A basically text format is based on Github Flavored Markdown.

The mdSlide has tiny change of horizontal ruler to create multi-page presentation. You can split page of presentation by ---. This format is similar to Deckset.

# Slide 1

foobar

---

# Slide 2

foobar
